Sitio web estático colaborativo con Terraform + GitHub Actions

Terraform y GitHub Actions en Azure

Descripción

Este proyecto crea un sitio web estático usando Terraform para gestionar la infraestructura en Azure y GitHub Actions para desplegarlo automáticamente.

Características

  • Infraestructura como código con Terraform
  • Automatización con GitHub Actions
  • Hosting estático en Azure
  • Colaboración mediante Git

Flujo de trabajo

1. Subo código a GitHub
2. GitHub Actions ejecuta el pipeline
3. Terraform crea o actualiza recursos en Azure
4. Sitio web se publica automáticamente

Tecnologías

Terraform, GitHub Actions, Azure, HTML y CSS.
